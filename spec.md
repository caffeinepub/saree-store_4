# Specification

## Summary
**Goal:** Replace the existing jewel-tone color scheme (deep crimson, gold, ivory) across Dali's Boutique with a fresh, elegant cool-toned palette (e.g., emerald and champagne).

**Planned changes:**
- Update all CSS custom property color tokens in `frontend/src/index.css` and `frontend/index.css` to the new palette
- Replace Tailwind config color tokens (gold, crimson, ivory, maroon) with the new palette values in `tailwind.config.js`
- Update all inline color references across components and pages: Header, Footer, HomePage, CategoryPages, OffersPage, ProductCard, CartDrawer, Chatbox, SuitabilityChecker, VirtualTryOn, AdminPage, and all other UI components
- Replace hero banner and category tile images with new cool-toned versions

**User-visible outcome:** The entire website reflects a cohesive new color palette, distinctly different from the previous crimson/gold/ivory theme, while all existing functionality remains intact.
